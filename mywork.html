MY WORK WITHIN WEB DESIGN

Below you find a few HTML assignments that were part of a website development course I participated. The aim of the course was to learn how to design and implement static websites by using HTML and CSS. I started from some very simple tasks like changing the font family or background color. I made progress and started learning more challenging tasks for example using Bootstrap and icons or making forms and buttons.

I am still at the very beginning of my web designer career, but I am very eager to learn more. My next goal is to learn the basics of JavaScript so I will be able to create dynamic webpages.
